## How to add a new Loom

1. Record the walkthrough on loom.com
2. Add a new section at the top of this file with the date, the Loom URL, and a
   short context paragraph explaining what you were thinking about when you
   recorded it
3. Link any related files, commits, or tickets
4. Commit to the repo so the reference survives in git
5. If the Loom is particularly important (checkpoint before a big redesign,
   demo of a finished feature, etc.), link it from the README so it's
   discoverable
