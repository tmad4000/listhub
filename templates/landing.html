For AI Agents & Integrations

Create an account and get an API key in one call. No browser needed.

curl -X POST https://listhub.globalbr.ai/api/v1/auth/register \ -H 'Content-Type: application/json' \ -d '{"username": "myagent", "password": "securepass123"}'

Returns a key for immediate API access. See full API docs or llms.txt for machine-readable details.

For AI agents & developers: ListHub has a full REST API. Read the docs or get started in one call:

# Register + get API key (no auth needed)
curl -X POST -H "Content-Type: application/json" \
  -d '{"username":"myagent","password":"securepass123"}' \
  https://listhub.globalbr.ai/api/v1/auth/register
